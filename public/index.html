Andrew's Directory App

Guide & Tutorial to this Express App:

This app supports the following functions:

    Register & Login Routes

  • Register as an Admin /register (POST) with first_name (string), last_name (string), email_address (string), password (string)
  • Login as an Admin /login (POST) with email_address (string), password (string)

    Company Routes

  • Create a Company /company (POST) with company_name (string)
  • Update Company Name /company/:company_id (PATCH) with company_name (string)
  • List Available Companies /company (GET)

    Create Admins for Specific Company

  • Invite an Admin /company/:company_id (POST) with email_address (string)

    Company Branches

  • Create a Company Branch /company/:company_id/branch (POST) with branch_name (string)
  • Update a Company Branch Name company/:company_id/branch/:branch_id (PATCH) with branch_name (string)
  • Delete a Company Branch company/:company_id/branch/:branch_id (DELETE)
  • List Company Branches company/:company_id/branch (GET)
  • Show Specific Company Branch company/:company_id/branch/:branch_id (GET)

    Employee Calls

  • List of All Users Available - /company/:company_id/branch/:branch_id/employee (GET)
  • Detailed Information About a Specific User - /company/:company_id/branch/:branch_id/employee/:id (GET)
  • To Create a New User - /company/:company_id/branch/:branch_id/employee (POST) with first_name (string), last_name (string), employee_email (string), job_title (string)
  • To Update an Existing User - /company/:company_id/branch/:branch_id/employee/:id (PUT) with first_name (string), last_name (string), employee_email (string), job_title (string), is_active (boolean)
  • To Delete an Existing/Specific User - /company/:company_id/branch/:branch_id/employee/:id (DELETE)
